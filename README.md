# Layout HTML + CSS

Responsive layout project built with HTML and CSS to practice semantic structure, mobile-first styling and page organization.

## Technologies
- HTML5
- CSS3

## Project structure
- `index.html` (main page)
- `index desktop.html` (desktop variant)
- `assets/css/style.css` (main styles)
- `assets/css/style-desktop.css` (desktop styles)
- `assets/css/reset.css` (reset styles)

## Main features
- Semantic HTML structure
- Responsive layout organization
- Mobile-first styling approach
- Separate desktop stylesheet
- Visual hierarchy, spacing and typography practice

## Project goal
Project created during the Front-End path at Epicode to practice HTML/CSS layout building and responsive design fundamentals.
